Comment utiliser

1. Inclure Font Awesome

<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.4.0/css/all.min.css">

2. Utiliser une icône

<i class="fab fa-instagram"></i> Instagram
Instagram

3. Personnaliser

.fab.fa-instagram {
    color: #E1306C;
    font-size: 2rem;
}
